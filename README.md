# ☕ SmartTask

Aplicación de consola desarrollada en Java para gestión de tareas.

---

## 📌 Funcionalidades

- Agregar tareas
- Listar tareas
- Marcar tareas completadas
- Eliminar tareas
- Manejo de prioridades
- Pruebas unitarias con JUnit5

---

## 🛠️ Tecnologías

- Java 17
- Maven
- JUnit5
- VSCode / Eclipse

---

## ▶️ Ejecutar

```bash
mvn clean install
java -jar SmartTask.jar
